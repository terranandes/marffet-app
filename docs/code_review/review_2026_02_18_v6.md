### 1. `scripts/ops/generate_ky_patches.py` — YFinance KY Patch Generator

**Verdict**: ✅ **Approved with Notes**

- Correctly identifies KY/DR/* stocks from the Excel reference file.
- Fetches `.dividends` (Cash) and `.splits` (Stock) from `yfinance`.
- Converts split ratio to stock dividend value: `stock_val = (ratio - 1) * 10`.
- Outputs `data/ref/ky_dividend_patch.json` with per-year cash/stock breakdown.

**Notes**:
- ⚠️ Split conversion formula `(ratio - 1) * 10` is correct for standard stock dividends (e.g. 1.1 → 1.0 stock div = 10%). However, for large par-value changes (e.g. 4763 split 10.0 → stock=90.0), this produces an extremely large stock dividend value that may distort CAGR calculations. **Recommend capping stock values at a reasonable threshold (e.g. ≤ 20.0)**.
- 152 stocks scanned, 90 with valid YFinance data (62 delisted/not found — expected).
