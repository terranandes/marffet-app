# Martian Web App (Frontend)

This is the React frontend for the Martian Investment System.

## Stack
- **Framework**: React + Vite
- **Styling**: TailwindCSS
- **Icons**: Lucide React
- **Charts**: Plotly.js (via `react-plotly.js`)
- **Components**: Framer Motion (animations), TanStack Table

## Getting Started

1.  **Install Dependencies** (if not already done):
    ```bash
    npm install
    ```

2.  **Start Development Server**:
    ```bash
    npm run dev
    ```
    Open [http://localhost:5173](http://localhost:5173).

## Features
- **Mars Strategy**: View Top 50 Low Volatility Stocks.
- **CB Strategy**: Real-time Arbitrage Calculator.
- **Visualization**: Bar Chart Race Demo.
